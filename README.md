# 🚌 Bus Reservation System

A modern **Full Stack Bus Reservation System** that allows users to search buses, view seat availability, book seats, and manage reservations through a simple and responsive interface.

---

## 🚀 Live Demo

🔗 http://localhost:5173/

---

## 📌 Features

- 🔐 User Authentication (Register & Login)
- 🚌 Browse available buses
- 📍 Search buses by source and destination
- 💺 Interactive seat selection
- 📅 View bus timings and travel details
- 💰 Booking summary with fare calculation
- 📱 Responsive UI
- 🔒 Secure API with JWT Authentication
- 🗄️ MongoDB database integration

---

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/bus-reservation-system.git
```

```bash
cd bus-reservation-system
```

---

### Install Frontend

```bash
cd client
npm install
npm run dev
```

---

### Install Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **server** folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```
